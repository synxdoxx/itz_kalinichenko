Игра "Доберись до учебы"
По сюжету пользователь просыпается по будильнику. Далее в зависимости от его выбора он может пойти по разным веткам. В ходе сюжета настает зомби апокалипсис, который настигает пользователя в разных местах в зависимости от ветки. 
Изначально предусмотрен определенный набор артефактов, который записывается в нужный файл при запуске игры. 
Артефакты выдаются или при победе в бою (при побеге от врага награды нет) или при определенных выборах по ходу прохождения сюжета.
Пользователь может создать свой аккаунт, данные об акаунтах тоже хранятся в отдельном файле. Для сохранений создаются песонализированные файлы для каждого такого аккаунта. Затем пользователь сможет зайти в старый аккаунт при следующих запусках кода.
Выборы пользователя влияют на концовку - есть несколько плохих и хороших, а также одна очень хорошая (и короткая).
Хорошие концовки также предполагают автоматическое сохранение при завершении работы игры.
Есть несколько вариаций функций сохранения и загрузки сохранений - пользователям доступны только по одной из каждого набора, остальные созданы под определенные ситуации. 
При проигрыше в бою игрок может попробовать сразиться снова сколько угодно раз либо закончить игру - тут и вступает в роль важность сохранений, так как несохраненные артефакты в любом случае возвращаются обратно в банк артефактов.
